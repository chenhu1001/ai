# Agent智能体
## 智能体-图形化
* Dify（图形化智能体平台）：提供可视化方式搭建 LLM 应用/智能体（工作流、RAG、工具调用）的低代码平台。

## 智能体-编码
* LangChain + LangGraph（智能体框架）：用于构建大模型应用与多智能体流程，其中 LangChain负责工具与链式调用，LangGraph负责状态机式复杂Agent流程编排。
* LlamaIndex（数据增强框架）：专注于把外部数据（文档、数据库、API）高效索引并接入大模型，用于构建RAG知识增强型应用。

## 多智能体
* AutoGen：微软开源的多智能体框架，用来让多个 LLM 自动对话协作完成任务。
* CrewAI：一个偏“团队工作流”的多智能体框架，通过角色分工（分析、执行、总结）来完成任务。

## 智能体决策机制
* ReAct：一种推理+行动结合的方法，让模型“思考一步→执行一步→再思考”的循环决策模式。
* Reflection（反思机制）：让模型在完成任务后回顾自己的输出与过程，找出错误并优化下一轮决策的迭代机制。
* Plan-and-Execute（规划-执行）：先由模型生成整体任务计划，再按步骤逐一执行并根据结果完成任务的分阶段决策模式。
